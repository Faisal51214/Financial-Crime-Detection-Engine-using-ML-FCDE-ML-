# Financial-Crime-Detection-Engine-using-ML-FCDE-ML-
Financial Crime Detection Engine using Machine Learning (FCDE-ML)  FCDE-ML is a lightweight AI system that scores financial transactions for potential fraud and AML risk. It combines supervised machine learning with synthetic risk labels to classify activity as Legitimate, Suspicious, or Likely Fraudulent.

## Features

- Synthetic but highly realistic AML transaction dataset  
- Rule + ML hybrid label generation  
- Data cleaning, preprocessing, feature engineering  
- Multiple models: XGBoost, Random Forest, MLP  
- Performance comparison and ROC AUC  
- Confusion matrices and classification reports  
- SHAP explainability for transparency  
- Threshold tuning for operations  
- End-to-end reproducible notebook included
